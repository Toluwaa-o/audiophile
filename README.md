**Audiophile E-commerce Website**

**Description:**
A responsive e-commerce web app where users can browse products, manage their cart, and complete checkout with form validation and order confirmation.

**Main Features:**

* Browse audio equipments by category
* View product details and images
* Add items to the cart

**Tech Stack:**

* **Next.js** (framework, routing, static generation)
* **TypeScript** (type safety)
